# Dulces Petalos App

## Una aplicación simple que muestra un listado de flores con una pantalla de detalle.

La app está desarrollada en Android nativo con el lenguaje Kotlin.

Componentes de desarrollo:
  - Arquitectura MVVM
  - Retrofit
  - Coroutines
  - Navigation
  - Safe args
  - Hilt
  - Glide
